Earthless / Destruction Unit / Yakuza / Running

This show at the Empty Bottle was phenomenal. All four of these bands put on great shows, and the Bottle is such a great venue. It had some weird sponsorship thing with Red Bull, so you had to RSVP to get in for $5, but it was first-come first-serve, no guaranteed ticket. So we got there a little before 9, and the first band didn’t start until 9:45.
Running was first, and I’m glad I got there to see them.

Tim Hecker / Oneohtrix Point Never

I got to see this show at Lincoln Hall, January 16th, 2014 as part of the Tomorrow Never Knows festival. I was unsure what to expect from this show as both these bands albums are kind of ambient electronic music. I’ve liked both of these albums a lot. I often listen to them through headphones at work. It’s a nice backdrop for getting work done, but there’s also interesting stuff going on in there if you want to listen intently.

Avocado

Pearl Jam’s new album is officially released, Tuesday May 2nd. I received my pre-ordered copy Friday. Thanks, Ten Club.
It’s a really good album. Upon first listen I thought Riot Act was a much better ablum, and wasn’t sure why this one was so over-hyped. Riot Act IS a great ablum, which doesn’t get the respect it deserves, but this self-titled album is really good too. It is growing on me.
