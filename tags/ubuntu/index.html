Airlink+ AWLL3025 on Ubuntu Linux 5.04

Note: I’m writing this mostly from memory, will go back and check, and update this to make sure it’s completely correct.
The Airlink+ AWLL3025 is a cheap 802.11g USB adapter.
Open Source Linux drivers for this adapter, however they aren’t usually included with any distro that I have seen, so you’ll need to compile them from source.
It seems the source files are not hosted on sourceforge anymore.
http://zd1211.ath.cx/ is the homepage for the zd1211 driver project.
